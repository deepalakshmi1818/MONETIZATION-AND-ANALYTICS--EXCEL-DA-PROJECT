# 📊 Project Overview
This project transforms raw digital video metrics into an interactive, decision-ready analytics dashboard. It covers the end-to-end data pipeline—from structural data cleaning and advanced formula modeling to dynamic business intelligence reporting. The primary objective is to evaluate current revenue frameworks, simulate progressive financial updates, and highlight market-leading creators.

---

## 🛠️ Key Technical Steps & Methodologies

### 1. Data Engineering & Cleaning
* **Data Sanitization:** Scrubbed the source dataset to remove anomalies, inconsistencies, and errors.
* **Clarity & Focus:** Hid non-essential background metrics to establish a high-density, high-impact workspace.

### 2. Advanced Formula Modeling
* **Data Synthesis (`VLOOKUP`):** Structured and cross-referenced relational data points across tables for seamless category mapping.
* **Dynamic Arrays (`UNIQUE`):** Programmatically isolated a clean, distinct master list of content categories.
* **Progressive Financial Adjustments:** Engineered calculations using custom scaling proportions to systematically simulate and evaluate revised monetization rates by category.
* **Audience Share Analysis:** Calculated precise percentage metrics to assess individual category view counts relative to the macro ecosystem.

### 3. Business Intelligence & Dynamic Summarization
* **Multidimensional Pivot Tables:** Built comprehensive summary engines to aggregate massive view totals, baseline rates, and simulated payout revisions seamlessly.
* **Top-10 Leaderboard Architecture:** Filtered and isolated the top 10 individual channels by traffic volume to recognize peak-performing, award-eligible creators.

### 4. Interactive Dashboard Implementation
* **Dynamic Slicers:** Connected a global, centralized Category filter across multiple background pivot components simultaneously.
* **Cohesive Visual Analytics:** Combined multi-axis analytical charts and contextual leaderboards into a clean, gridline-free user interface.
* **Interactive Synchronization:** Enabled stakeholders to dynamically toggle across content niches, causing all monetization trends, charts, and leaderboards to update in real time.
